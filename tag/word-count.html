一：实验目标

  • 学习docker基本命令

  • 搭建只有一个master和两个slave的Hadoop集群环境

  • 进行wordcount实验,理解mapreduce和hdfs等工作机制

二：docker基本命令

docket是目前较为流行的虚拟化技术，可以方便轻松的在Linux中建立容器，达到虚拟化需求。

基本操作命令如下，更多参考这里。

  • 启动容器：docker run 镜像名

  • 暂停容器：docker stop 容器名

  • 重启开启容器：docker start 容器名

  • 进入容器：docker attach 容器名
